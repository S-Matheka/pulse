## Prerequisites

- Node.js (v16 or higher)
- npm or yarn

## Getting Started

1. Clone the repository:
```bash
git clone [repository-url]
cd healthcare-pulse
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm run start
# or
yarn start
```

The application will open in your default browser at `http://localhost:3000`.
